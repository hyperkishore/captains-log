2 One-Line Installation

Open Terminal and run this command:

Terminal
curl -fsSL https://raw.githubusercontent.com/hyperkishore/captains-log/main/installer/install.sh | bash

3 What Gets Installed

The installer will:

  • Install Homebrew & Python (if needed)
  • Create isolated Python environment
  • Install Captain's Log from GitHub
  • Set up auto-start via launchd
  • Install Node.js & React dashboard
  • Create CLI wrapper captains-log

4 Grant Accessibility Permission Required

Accessibility Permission Required

Captain's Log needs Accessibility permission to read window titles. Without it, only app names will be tracked.

1

Open System Settings

The installer will open this automatically, or go to System Settings ‚Üí Privacy & Security

2

Navigate to Accessibility

Click Accessibility in the left sidebar

3

Enable Terminal

Find Terminal (or your terminal app) and toggle it ON

4

Restart the daemon

Run: captains-log stop && captains-log start

5 Verify Installation

1

Check daemon status

Run: captains-log status

2

Run: captains-log health

3

Run: captains-log-dashboard then open http://localhost:3000
